# Architecture Plates

This directory contains the official Project Keystone Architecture Plates.

Architecture Plates are first-class visual artifacts that communicate the architecture, capabilities, and executive reasoning models of Project Keystone.

Every plate follows the Project Keystone Visual Design Language and Architecture Plate standards.
